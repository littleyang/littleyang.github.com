## 第一个程序:Hello,world。

    hellowolrd程序，可谓是世界上最完美的一个程序，对于你学习任何语言，都是了解入门的第一步。当然，我们只是需要
通过这个程序输出"hello,world"，来表达这种语言的执行结果,显示语言的特性。代码如下:

```python
print "hello,world"

```

执行结果:

```
hello,world

```
